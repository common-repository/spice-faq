== Description ==

FAQ plugin that lets you create, organize and publicize your FAQs (frequently asked questions) in no time through your Wordpress admin panel. You can show FAQ on your site with the help of *[spicefaq]*
FAQ can be displayed using a shortcode in post, pages and sidebars.

Here is the link to [Documentation](https://spicethemes.com/documentation-for-spice-faq-plugin/)

== Installation ==

1. Download Spice FAQ plugin.
2. Unzip and upload the spice-faq folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the 'Plugins' menu in WordPress and Enjoy.
